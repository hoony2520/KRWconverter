How to use.

You can get a JSON object when you make an AJAX request to currency.php.
'currency.php' and 'simple_http_dom.php' should be in same directory.


사용법:

currency.php로 AJAX요청을 보내면 응답을 JSON 형태로 반환합니다.
currency.php와 simple_http_dom.php 파일은 동일 위치에 있어야 됩니다.
